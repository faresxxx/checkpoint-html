Introduction

Un site web est un ensemble de pages web et des ressources inter reliées entre elles par des liens hypertextes, auxquelles l'internaute peut accéder par une adresse web appelée URL. Un site web ou site internet englobe des textes et multimédia.

 Une page web (ou document HTML) est un fichier texte écrit en langage HTML et enregistré avec l’extension .html

 Le langage HTML décrit la structure (le squelette) d'une page web.

 Le langage CSS sert à la mise en forme et la mise en page d'une page web. Ce langage est essentiel pour embellir et donner une apparence attractive à une page web.

Un éditeur web : sert à écrire le code des pages web (Visual Studio Code, Dreamweaver, Notepad++…)

une balise HTML:

 est l’ensemble des éléments de base du codage HTML. Elle est utilisée pour mettre en forme un texte, structurer et hiérarchiser le contenu d'une page. Les balises indiquent également au navigateur comment afficher la page en question. Pour l'internaute, elles sont invisibles sauf s'il affiche le code source.

Qu'est-ce qu'une balise en HTML ?

Le HTML repose sur une syntaxe simple et rigide, dont l'unité de base est la balise.Les balises vont toujours par deux, une pour ouvrir l'élément, l'autre pour le fermer. Elles se construisent toujours de la même façon:

  1. un chevron < pour ouvrir la balise
  2. Le nom de l'élément contenu dans la balise
    3. Un attribut, si nécessaire, pour qualifier l'élément. Par exemples la source ou la balise alt d'une image. Pour ajouter l'attribut, à l'intérieur de la balise : espace, nom de l'attribut (src, alt, etc), signe égal (=) et une valeur s'il y a lieu, placée entre double quotes (""). Exemple: un chevron > pour fermer la balise

La balise d'ouverture contient le nom de l'élément et l'attribut. Une balise de fermeture termine l'élément, elle ne reprend que le nom de l'élément accompagné d'un slash (/). Par exemple, la balise paragraphe :

L’anglais « Hypertext Markup Language » se traduit littéralement en « langage de balisage d’hypertexte »1. On utilise généralement le sigle « HTML », parfois même en répétant le mot « langage » comme dans « langage HTML ». Hypertext est parfois écrit HyperText pour marquer le T du sigle HTML. Le public non averti parle parfois de HTM au lieu de HTML, HTM étant l’extension de nom de fichier tronquée à trois lettres, une limitation que l’on trouve sur d’anciens systèmes d’exploitation de Microsoft.
